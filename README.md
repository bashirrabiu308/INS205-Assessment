# INS205 Assessment – Personal Profile Web Page

## Overview

This project was developed as part of the INS205 Continuous Assessment at the University of Jos.

The project is a static web application built using HTML, CSS, and JavaScript. It presents personal information, educational background, skills, interests, and contact details in a clean and responsive web page.

## Features

* HTML5 compliant structure
* Responsive page layout
* Professional user interface using CSS
* About Me section
* Educational Background section
* Skills section
* Interests and Hobbies section
* Contact section
* Functional "Send Email" button using the `mailto:` protocol
* Dynamic button colour generated from matriculation number using JavaScript

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open `INS205_Assessment.html` in any modern web browser.
3. Click the **Send Email** button to contact the author through the provided university email address.
